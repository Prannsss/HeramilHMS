# **App Name**: HeramilHMS

## Core Features:

- Secure Login: User authentication and role-based access control for admins and doctors.
- Availability Display: Display of doctors' availability schedule. Indicates open appointment slots. Uses a tool to auto-adjust schedule in case of known, predictable events like holidays.
- Patient Info Forms: Allows user to enter/update basic details such as name, age, gender, and contact information.
- Digital Records: Provides digital displays of prescriptions and test results, allowing downloads of PDF.
- Automated Reporting: Generates reports about service logs and overall hospital usage statistics.
- Appointment Management: Streamlines and displays key appointment details such as time, doctor, and reason, status.
- AI Diagnosis Support: AI powered diagnosis tool to use information given to help determine what possible diseases the patient may have. The tool helps with differential diagnosis suggestions based on symptoms.
- Admin Dashboard: Dedicated dashboard for admin users to manage staff, patients, appointments, billing, and reports.
- Doctor Dashboard: Dedicated dashboard for doctors to view assigned appointments, add diagnoses, prescriptions, and test results.
- Inventory: Inventory to store the medical supplies

## Style Guidelines:

- Primary color: HSL 210, 60%, 50% - RGB(38, 127, 221) - A calming blue to inspire trust and reliability.
- Background color: HSL 210, 20%, 95% - RGB(242, 247, 252) - A light, desaturated blue for a clean, professional feel.
- Accent color: HSL 180, 50%, 40% - RGB(51, 153, 153) - A contrasting cyan-green to highlight important actions and elements.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text.  Its modern and neutral design offers readability and a professional aesthetic.
- Simple, clear, and consistent icons that represent different features (e.g., calendar for appointments, user for patient profiles).
- Clean and structured layout with a focus on readability and easy navigation. Prioritize essential information with a minimalist design.
- Subtle animations for loading states and transitions to provide feedback without being distracting.
